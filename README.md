# Fun Investor Bot

Telegram бот для мотивации инвестиций и финансового планирования. Помогает пользователям осознать, сколько денег они тратят на вредные привычки и как эти деньги можно инвестировать.

## Возможности

- Расчет ежемесячных трат на вредные привычки
- Конвертация валют
- Генерация мотивационных сообщений с использованием GPT
- Интерактивный диалог с пользователем

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/BilboBag77/Fun-investor.git
cd Fun-investor
```

2. Создайте виртуальное окружение и активируйте его:
```bash
python -m venv venv
source venv/bin/activate  # для Linux/Mac
venv\Scripts\activate     # для Windows
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

4. Создайте файл `.env` и добавьте необходимые переменные окружения:
```env
BOT_TOKEN=your_telegram_bot_token
OPENAI_API_KEY=your_openai_api_key
```

## Использование

1. Запустите бота:
```bash
python main.py
```

2. Откройте Telegram и найдите своего бота
3. Начните диалог, следуя инструкциям бота

## Разработка

### Структура проекта

- `main.py` - точка входа в приложение
- `handlers.py` - обработчики сообщений
- `calculator.py` - логика расчетов
- `gpt.py` - интеграция с GPT
- `config.py` - конфигурация приложения

### Тестирование

```bash
# Запуск тестов
pytest

# Проверка стиля кода
flake8
```
